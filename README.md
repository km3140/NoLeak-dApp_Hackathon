### 🔒 NoLeak 
<br>

‘NoLeak’ 는 횡령방지를 위한 **가상자산 공동관리** 솔루션으로 <br>
'n명' 이 사용하는 공동금고를 생성하여 코인(토큰)을 관리하는 **Multisig dApp** 해커톤 프로젝트 입니다

금고에 등록된 n명의 승인을 모두 받아야 금고에서 **출금 & 송금**을 할 수 있습니다
